# Numerical T Gate Reduction BQSKit Pass

This software implementation the awesome Numerical T Gate Reduction(link) work by
Marc Davis as an extension pass in BQSKit.

## Installation

This is available for Python 3.8+ on Linux, macOS, and Windows.

```sh
pip install ntro
```

## Import Circuits from External Libraries

BQSKit comes with extension support for Qiskit, TKet, Cirq, and Qutip.
To convert circuits between those libraries and BQSKit, import any of
the follow functions from the `bqskit.ext` module:

```
qiskit_to_bqskit
qutip_to_bqskit
cirq_to_bqskit
pytket_to_bqskit
bqskit_to_qiskit
bqskit_to_qutip
bqskit_to_cirq
bqskit_to_pytket
```
